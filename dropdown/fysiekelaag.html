Fysieke Laag

De fysieke laag is de basis van het drie-lagenmodel. Hier vind je alle hardware die mijn computer nodig heeft om te werken. Het zorgt ervoor dat informatie kan worden verzonden en ontvangen. Hier zijn de belangrijkste onderdelen van de fysieke laag:

Processor

De processor, ook wel CPU genoemd, is het brein van mijn computer. Hier worden alle berekeningen en beslissingen gemaakt. De processor voert opdrachten uit van programma's en zorgt dat alles soepel draait.

Kloksnelheid

De kloksnelheid geeft aan hoe snel mijn processor werkt. Dit wordt gemeten in gigahertz (GHz). Hoe hoger de kloksnelheid, hoe sneller mijn processor taken kan doen. Een processor van 3 GHz kan bijvoorbeeld drie miljard opdrachten per seconde aan.

Verwerkingssnelheid

De verwerkingssnelheid is hoe snel mijn processor gegevens verwerkt. Dit hangt af van de kloksnelheid Ã©n het aantal kernen. Een processor met meerdere kernen kan meerdere dingen tegelijk doen, wat mijn computer sneller maakt.

Geheugen: Permanent

Permanente geheugen slaat gegevens op, zelfs als mijn computer uitstaat. Hiermee bewaar ik programma's en bestanden. Voorbeelden hiervan zijn een harde schijf of SSD. Dankzij dit geheugen kan ik mijn bestanden terugvinden als ik mijn computer opnieuw opstart.

Randapparatuur

Randapparatuur zijn apparaten die ik aansluit op mijn computer, zoals een toetsenbord, muis, printer of scanner. Deze apparaten helpen mij om gegevens in te voeren of om informatie te ontvangen. Zonder randapparatuur kan ik niet goed met mijn computer werken.
